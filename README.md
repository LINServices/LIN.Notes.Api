# LIN Notes

Servidor de Notas de LIN.

* Crear notas.
* Notas de colores.
* Guardar datos en la base de datos.


### Endpoints

#### Authentication

Para la autenticación se usa el servicio LIN Identity.

* Login con credenciales (Usuario y contraseña).
* Login con token de acceso.


#### Notas

Acciones y métodos relacionados con las notas.

* Crear nota.
* Obtener notas asociadas a un perfil.
* Obtener una nota.
* Actualizar nota.
* Actualizar color de una nota.
* Eliminar nota.


Además también contiene soporte para actualización en tiempo real.
